# Quiz de Harry Potter - Android

Aplicativo Android desenvolvido em Java que simula o teste de seleção das casas de Hogwarts. O usuário responde a uma sequência de perguntas e, ao final, recebe a casa com a qual possui maior afinidade de acordo com suas respostas.

## Funcionalidades

- Cadastro do nome do usuário
- Quiz com perguntas de múltipla escolha
- Sistema de pontuação para cada casa
- Resultado final baseado nas respostas
- Interface intuitiva para navegação entre as telas
- Exibição da casa correspondente ao perfil do usuário

## Tecnologias utilizadas

- Java
- Android Studio
- XML
- Android SDK

## Estrutura do projeto

- Tela inicial
- Cadastro do usuário
- Sequência de perguntas
- Sistema de cálculo da pontuação
- Tela de resultado

## Objetivo educacional

Este projeto foi desenvolvido como atividade escolar com o objetivo de praticar conceitos do desenvolvimento Android, incluindo:

- Desenvolvimento de interfaces com XML
- Programação orientada a objetos em Java
- Navegação entre Activities
- Manipulação de eventos
- Lógica de programação
- Organização de projetos Android
